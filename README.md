# Customer Query & Order Tracking System (Excel-Based)

## Overview
This project is an Excel-based system used to manage customer queries, order tracking, and operational workflow efficiently.

## Features
- Customer Query Tracking
- Order Status Management
- Issue Escalation Tracking
- Data Accuracy Handling
- Excel Filters & Reporting

## Tools Used
- Microsoft Excel
- Data Validation
- Conditional Formatting
- Filters & Sorting
